# Last War - Season 6 Guide

## 🇸🇦 العربية

دليل شامل متعدد اللغات لشرح الموسم السادس (الغابة المفقودة) في لعبة Last War.

### المميزات

- 🌐 دعم ثلاث لغات: العربية، الإنجليزية، الصينية — مع تبديل فوري
- 📱 تصميم متجاوب بالكامل لجميع الأجهزة (ديسكتوب، تابلت، جوال)
- 📂 قائمة جانبية (Side Drawer) على الجوال مع تأثيرات انزلاق سلسة
- 🔄 حفظ اللغة المختارة تلقائيًا عبر localStorage
- 🎨 واجهة بتصميم داكن مع تأثيرات Glow خضراء

### محتوى الصفحة

- نظرة عامة سريعة عن الموسم السادس
- قصة الموسم والأحداث المرتبطة بالغابة المفقودة
- شرح لرسائل القصة والمهمات
- تحليل جدول أحداث الموسم السادس الرسمي
- توضيح الموارد الجديدة والمباني الجديدة
- عرض مكافآت الموسم والمتاجر الخاصة به
- لمحة عن المهن وخريطة الموسم

### التقنيات

- HTML / CSS / JavaScript
- Font Awesome 6.4.0
- تصميم RTL/LTR تلقائي حسب اللغة

### الهدف

تقديم شرح واضح وسريع عن الموسم السادس في صفحة واحدة سهلة التصفح، متاحة بثلاث لغات.

- 📂 Side drawer menu on mobile with smooth slide animations

- 🎨 Dark-themed UI with green glow effects

### Page Content

- Quick overview of Season 6
- Season story and Lost Rainforest events
- Story messages and missions explained
- Official Season 6 event timeline breakdown
- New resources and buildings explained
- Season rewards and special stores
- Professions overview and season map

### Tech Stack

- HTML / CSS / JavaScript

Provide a clear, fast guide to Season 6 on a single easy-to-browse page, available in three languages.
